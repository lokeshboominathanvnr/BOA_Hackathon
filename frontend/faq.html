User can save from short saving to fulfil a goal (Annual vacation, Car insurance renewal, Repayment of Loan..) by setting a goal amount and save continuously on all type of purchaces made through Debit card/internet/mobile banking.

User can select the option for "Goal-Based Small saving" setup option from Home page of mobile banking/internet banking application.

User can set a Goal amount(eg 1000 or 2000 or 5000). If the Goal amount is higher, user can select maximum percentage of deduction so that Goal Amount can be achieved quicker.

On usual saving, user must save some fixed amount fro a definite period. But, Goal based small savings gives the option to save a very minimal amount on every purchase without any restriction.

The user will get a notification as well as the Goal amount along with the interest earned on that will be credited to Saivng account when goal amount is reached.
